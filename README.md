# warez.rip

OpenCart extensions and themes safety checker.

## About

The [Warez R.I.P](https://warez.rip) service helps identify potentially unsafe, pirated, or suspicious OpenCart marketplaces distributing modules and themes.

The project maintains a curated database of known marketplaces and provides reputation-based safety information for OpenCart developers and store owners.

## Features

- Marketplace reputation lookup
- Detection of known warez marketplaces
- Trusted marketplace recommendations
- Guidelines on how to distinguish warez vs legitimate sources
- Information about risks and consequences of using pirated extensions
- Shareable search links (e.g. https://warez.rip/?q=example.com)
- Multilingual interface
- Dark and light theme support
